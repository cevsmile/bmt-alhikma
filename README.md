#### :house: Local web server 
[XAMMP][xampp] > `PHP My Admin` Or use navicat [BMTAL-Hikma.Versi.Alfa2.0.sql][database]

#### :name_badge: [CodeIgniter][ci]
This project already use [CI 2.1.4][ci]([CodeIgniter_2.1.4][ci]).
